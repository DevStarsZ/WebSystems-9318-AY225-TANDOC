# JPCS Student Organization Website

A modern, futuristic website for the Junior Philippine Computer Society (JPCS) with interactive features and mobile-responsive design.

## Features

### Core Features
- **Single Page Application (SPA)** with smooth transitions
- **Responsive Design** for all screen sizes
- **Modern, Techy UI** with neon effects and animations
- **Six Complete Pages**: Home, About, Officers, Events, Membership, Contact

### JavaScript Functionalities
1. **Event Filtering System** - Filter events by date, category, and search
2. **Membership Registration Form** - Multi-step form with validation
3. **Interactive Officer Profiles** - Clickable profiles with modal popups
4. **Event Details Modals** - Popup modals for detailed event information
5. **LocalStorage Usage** - Save form progress, preferences, and user data

### Pages Included
1. **Home** - Hero section, statistics, featured events
2. **About** - Mission, vision, history timeline, core values
3. **Officers** - Executive board, committee heads, member directory
4. **Events** - Event calendar, filtering, registration
5. **Membership** - Tier selection, registration form, benefits
6. **Contact** - Contact form, social links, FAQ, location
